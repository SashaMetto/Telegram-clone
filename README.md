# Telegram-clone

Telegram-clone is project made with MERN stack.

## Run Locally

Clone the project

```bash
  git clone https://github.com/SashaMetto/Telegram-clone
```

Go to the project directory

```bash
  cd Telegram-clone
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the Client

```bash
  //open terminal
  npm start
```
